# Lumen with Vue.js Example

## Project setup

### setup lumen
- in back folder 
- create `.env` file and add yor DB Config
- `composer install`
- `php artisan migrate`
- `php artisan DB:seed` (seed users in the table) 


### setup Vue
- in front folder 
- create `.env` file and but the backend url in `VUE_APP_BACKEND_URL` 
- npm install
- npm run serve (start Vue.js app)
